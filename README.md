## …or create a new repository on the command line

```

echo "# 4geeks-clase4" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M master
git remote add origin https://github.com/notanumber93/4geeks-clase4.git
git push -u origin master

```

```git remote add origin https://github.com/notanumber93/4geeks-clase4.git
git branch -M master

```
